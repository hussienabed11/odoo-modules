# Courses Manager (Odoo 17)

## Overview
A custom Odoo module designed to manage courses, instructors, students, and enrollment.

## Features
- Course model with basic fields (name, description, duration)
- Instructor model with relations
- Student model with tracking
- One-to-many and many-to-many relations
- Custom views (tree, form, search)
- Domain filters
- Security rules via access control list

## Technical Stack
- Python (Odoo ORM)
- XML Views
- Odoo Models & Fields
- Security (ACL)
- QWeb (optional)
